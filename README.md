# kNN-based-Recomendation-System-for-Educational-Courses
Рекомендательная система образовательных курсов на основе алгоритма k-ближайших соседей (k-NN).

## О проекте

Представляет собой **гибридную рекомендательную систему** для образовательных курсов, которая использует:
- **Взвешенный k-NN** для поиска похожих курсов
- **Категории навыков**
- **Стратегию 7 k-NN + 1 популярный** для рекомендаций
- **Холодный старт** для новых пользователей

---
## Система построена на навыках
Навыки содержатся в датасете и используются исходной системой Сoursera как ключевые слова для поиска курса. Например, Python, C++, HTML и так далее.

Данная система использует эти навыки, скомпоновааные по смыслу в категории, такие как Data Science, содержащие Python, Data Preprocessing и другие. k-NN использует каждую категорию как признак, каждый навык у подданного на вход курса дает свой вклад в признаковое пространство.
## Архитектура

```
┌─────────────────────────────────────────────────────┐
│                         КЛИЕНТ                      │
│                    (HTML / JavaScript)              │
└───────────────────────────┬─────────────────────────┘
                            │ POST /recommend
┌─────────────────────────────────────────────────────┐
│                        FASTAPI                      │
└───────────────────────────┬─────────────────────────┘
                            │
┌─────────────────────────────────────────────────────┐
│                      РЕКОМЕНДАЦИЯ                   │
│   ┌─────────────────────────────────────────────┐   │
│   │  1. Вектор пользователя - среднее курсов    │   │
│   │  2. Поиск 12 ближайших соседей (cosine)     │   │
│   │  3. Взятие 7 лучших                         │   │
│   │  4. 1 случайный из топ-15 популярных        │   │
│   │  5. Возврат 8 рекомендаций                  │   │
│   └─────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────┘
```

## Особенности

| Компонент | Описание |
|-----------|----------|
| **34 категории** | Навыки сгруппированы в осмысленные кластеры |
| **Счетчики, не бинарные** | 3 навыка Python → оценка 3, а не 1 |
| **Взвешенные признаки** | Категории 7.0x, Сложность 2.5x |
| **Стратегия 7 + 1** | 7 k‑NN + 1 случайный популярный |
| **Холодный старт** | 8 случайных популярных курсов для новых пользователей |

## API Эндпоинты

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| `GET` | `/` | Веб-интерфейс |
| `POST` | `/recommend` | Получить рекомендации |

### Пример запроса

```json
POST /recommend
{
  "course_titles": [
    "Crash Course on Python",
    "Python Basics"
  ]
}
```

### Пример ответа

```json
{
  "recommendations": [
    {
      "course_title": "Python Data Structures",
      "course_organization": "University of Michigan",
      "course_difficulty": "Beginner",
      "course_rating": 4.9,
      "source": "k-NN"
    }
  ]
}
```
